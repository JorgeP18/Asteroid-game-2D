# Asteroid-game-2D
Este es un proyecto dual donde me sirvió para aprender sobre las herramientas que ofrece unity. El juego se basa en que manejamos una nave espacial y tenemos que sobrevivir destruyendo asteroides, dentro del juego tenemos un sistema de vidas y salud donde se vincula todo con una interface que aparecera con los botones de reintentar o salir
